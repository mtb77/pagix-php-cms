Pagix Kontext-Hilfe

Falls Sie einen neuen editierbaren Seitenbereich anlegen, geben Sie zunächst unter "Seitenbereich" die genaue Bezeichnung des Platzhalters ein, wie er in der html-Vorlage benannt ist. Weiterhin geben Sie die maximale Anzahl der möglichen Objekte des Seitenbereichs an - wenn Sie "0" angeben, bleibt die Anzahl undefiniert. Falls Sie den Objekttyp "Bild mit Auto-Größe" verwenden wollen, geben Sie hier die gewünschte Bildgröße bzw. maximale Bildgröße an. Nach der Bestätigung können Sie nun die einzelnen Objekte des Seitenbereichs definieren. Wählen Sie dazu aus dem Drop-Down-Menü den gewünschten Objekttyp und klicken dann auf "Neues Objekt". Das neue Objekt erscheint in einer Liste. Hier müssen Sie nun angeben, wie oft ein Objekt dieses Objekttyps in dem editierbaren Seitenbereich maximal vorkommen soll. Außerdem müssen Sie aus dem Drop-Down-Menü "Vorlage" die gewünschte Objekt-Vorlage dem Objekt zuordnen. In den Objekt-Vorlagen hat der Designer das Erscheinungsbild für die Objekte festgelegt, z.B. die Farbe, Größe und Schriftart von Text, oder die Größe und den Freistellraum von Bildern. Nach dieser Zuordnung können Sie weitere Objekte anlegen. Mit "zurück" kommen Sie auf die Übersicht der editierbaren Seitenbereiche. Auch hier können Sie nun weitere editierbare Seitenbereiche - wenn in der Vorlage so vorgesehen - nach dem geschilderten Schema definieren. Nach diesen Definitionen können auf Basis dieser Vorlage Seiten erstellt und mit Inhalt gefüllt werden. Wenn Sie einen schon bestehenden editierbaren Seitenbereich verändern wollen, so ändern Sie die bisherigen Einstellungen so, wie Sie es wünschen. Vergessen Sie nicht, jede Änderung mit dem jeweiligen "Übernehmen"-Button zu bestätigen.
