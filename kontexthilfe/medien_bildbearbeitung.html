Pagix Kontext-Hilfe

JPG-Bilder bearbeiten: Bilddateien (jpgs) können Sie bearbeiten (Ausschnittanpassung, Skalierung), indem Sie im Verzeichnisbaum auf das Bild klicken. Das Bild erscheint in einem Bearbeitungsfenster. Dort können Sie es auf die gewünschte Größe skalieren. Durch Eingabe der Zielbild Höhe und Breite sowie durch Anwahl der Buttons "größer" oder "kleiner" legen Sie den gewünschte Ausschnitt bzw. die gewünschte Ausschnittsgröße fest. Das Auswahlfenster bewegen Sie mit den vier Buttons in der eingestellten Schrittweite. Das Ergebnis können Sie sich durch klicken auf die "Vorschau" ansehen. Wenn Ihnen das Ergebnis zusagt, wird das Bild durch das bearbeitete ersetzt, indem Sie auf "Speichern" klicken.
