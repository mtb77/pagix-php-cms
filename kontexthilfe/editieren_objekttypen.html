Pagix Kontext-Hilfe

Objekte sind Platzhalter für Inhalte, die Sie selbst eingeben oder austauschen. Inhalte können z. B. sein: Texte, Bilder, Überschriften, Zwischenüberschriften, Interne Links (Verzweigungen), Externe Links (Verzweigungen), Media-Links (Verzweigungen auf Medien).
Wenn Sie ein neues Objekt einfügen wollen, dann wählen Sie unter "Neues Objekt einfügen" per Drop-Down-Menü den gewünschten Objekttyp aus (falls mehrere Objekttypen möglich sind) und klicken dann auf "Übernehmen".
Bei "Text" (und auch "Überschriften" und "Zwischenüberschriften") geben Sie den gewünschten Text in die Maske ein und klicken auf "Übernehmen".
Bei "Interner Link" können Sie per Drop-Down-Menü die Seite bestimmen, auf die verzweigt werden soll, wenn der Linkname, den Sie vergeben, angeklickt wird. Sie können mit der Option "Link in neuem Fenster öffnen" beim Anklicken das Öffnen eines neuen Fensters erzwingen. Ansonsten wird in das bestehende Fenster verlinkt. Bestätigen Sie Ihre Eingaben wiederum mit "Übernehmen".
Das Medium "Bild" setzt voraus, dass das entsprechende Bild schon auf den Server geladen wurde. Sehen Sie dazu unter der Rubrik "Medien" nach. Wiederum per Drop-Down-Menü können Sie aus der Liste der zur Verfügung stehenden Bilder auswählen, nachdem der gewünschte Verzeichnispfad ausgewählt wurde. Weiterhin besteht die Möglichkeit, das ausgesuchte Bild mit einer Verzweigung (Link) zu versehen, um bei Klick darauf zu einer anderen Seite zu gelangen - wählen Sie dazu die entsprechende Option "Ja". Einen Link ausserhalb Ihrer Internetpräsenz müssen Sie direkt eingeben (in der Form: "http://www.domain.de/seite.html"). Eine Verlinkung innerhalb Ihrer Site können Sie per Drop-Down-Menü auswählen. Bestätigen Sie Ihre Auswahl mit "Übernehmen.
Beim Objekttyp "Media Link" geben Sie zunächst den Text des Links ein. Mittels Drop-Down-Menüs wählen Sie dann Verzeichnispfad und Dateiname des Bildes aus. Gegebenenfalls wählen Sie noch die Option "Link in neuem Fenster öffnen" und bestätigen mit "Übernehmen".
Den Objekten sind jeweils die Optionen "Löschen" und "Verschieben" zugeordnet. Mit "Löschen" entfernen Sie das zugeordnete Objekt". Mit "Verschieben" können Sie die Reihenfolge der Objekte verändern, wenn der Platzhalter (editierbarer Seitenbereich) mehr als nur ein Objekt beinhaltet.
Wenn Sie auf "Seitenvorschau" klicken, wird Ihnen das Ergebnis Ihrer Eingaben angezeigt. Aus technischen Gründen kann diese Darstellung aber noch von der späteren tatsächlichen Darstellung abweichen, wenn die neue Seite im Kontext von anderen Seiten angezeigt wird.
Nach allen Eingaben können Sie mit "zurück" wieder zur Übersicht der editierbare Seitenbereiche der Seite gelangen.
